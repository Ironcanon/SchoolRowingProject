The login screen

This is the opening screen of the program, to access the rest of the program you will have to enter a valid username and password. You should have been given a valid username and password by your school's administrator or head coach. If you are struggling to login ensure that you are entering the correct username and password combination. If the issue persists please inform your school's administrator or head coach.

The dashboard screen

This is the main screen of the program as it is where the GUI, the visual representation of the trailer layout, lives. The user can't actually do very much on the dashboard screen but they can access every other part of the program from here including the view boat screen, the set boat screen and the swap boat screens. If the GUI of the program is empty it means that the admin of the school the user is currently signed in under has either not yet placed any boats in the trailer or that the school has not entered any boats into the database.

The add boat screen

This is where admins can add boats to their school's database. To successfully input a boat into the database it must pass a validation test which is performed on all of the fields:
Firstly all the fields must have data inputed, whether it is a text field or a radio button selection, then there are specifics for each data piece entered.

  • For boat names they can not contain numbers but can contain symbols.
  • For the number of seats it has to be a number that is either 1,2,4 or 8.
  • For the HIN of the boat it has to have a length of 12 and the first 2 characters have to be characters
  • For the oar type it only has to have one selected
  • For the purchase date it must be in the past

The view boat screen

This screen is where all users can view the boats in the current school's database and admins can also edit and delete boats. If the edit and delete buttons are greyed out this means that the current user is not an admin and so can not edit or delete boats. The list at the top of the screen should contain all the boats in the current user's school's database, if it is empty then the school currently has no boats in the database.

The set boat screen

This is where admins set boats in the database to slots in the trailer GUI. If the set boat combo box is empty then there are not any boats in the database for the user's school

The swap boat screen

This is where admins can swap boats that are already in the trailer GUI. If there is only one boat in the GUI it will be the only boat in both combo boxes, otherwise it will show all the boats in the trailer GUI in both combo boxes.
